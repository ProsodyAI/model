# ProsodyAI Model (@prosodyai/model)

Core prosody/emotion model library (ProsodySSM). Pip package: **prosody-ssm** (import `prosody_ssm`).

- **Repo**: [ProsodyAI/model](https://github.com/ProsodyAI/model)
- **Install**: `pip install prosody-ssm` or `pip install prosody-ssm @ git+https://github.com/ProsodyAI/model.git@main`

Part of the [ProsodyAI](https://github.com/ProsodyAI/prosodyai) monorepo as a submodule at `prosody_ssm/`.
